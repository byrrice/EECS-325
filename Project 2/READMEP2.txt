README for Project 2

Alright, step by step instructions for future people, aka me:
1. In putty, put IP Address of your virtual machine in the hostname
2. Click Open, then login with your username and password
3. mkdir project2
4. cd project2
5. To install matplotlib or any library, pip3 install NAMEOFLIBRARY
6. For the love of god, use recvfrom instead of recv
7. sudo python3 distMeasurement.py
8. cat results.txt/whatever results file is
8.5 if extra credit, pip3 install geoip2
8.6 sudo python3 geoDistance.py
8.7 cat distance_results.txt (will be generated despite timeout error) 
8.75 pray
10. Do this for project2grading

To put files in server (DO THIS FOR THE GEOLITEMMDB FILE TOO):
1. psftp IP_ADDRESS
2. Enter login info
3. cd project2
4. put LOCATIONOFFILE

In general, run using sudo python3 FILENAME.py
